# Agent Guide

Use `CLAUDE.md` as the main agent guide for this repository.

When working on `content_inbox/`, also read `content_inbox/CLAUDE.md`.

Do not read `.env`, secrets, tokens, credentials, keys, `content_inbox/data/**`, or `content_inbox/outputs/runs/**`.

Before changes, provide a short plan. After changes, report files changed, verification commands, risks, and any manual confirmation needed.
